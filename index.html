Pentru a curăța codul de codurile CSS nefolosite în HTML, trebuie să îndepărtezi toate regulile CSS care nu sunt utilizate în structura HTML a paginii. Din fericire, există unele instrumente online și plugin-uri pentru editorul tău de cod care pot face acest lucru automat. Dacă vrei să faci acest lucru manual, trebuie să analizezi fiecare selector CSS și să vezi dacă este utilizat în structura HTML. Poți face acest lucru folosind instrumente de dezvoltare web din browser (cum ar fi Chrome Developer Tools) pentru a inspecta fiecare element HTML și pentru a verifica dacă sunt aplicate regulile CSS. După ce ai eliminat codurile CSS nefolosite, codul curățat ar trebui să arate aproximativ așa: ```html Sephiroth's Gallery

Sephiroth
